Conoce nuestros servicios

Pintado

Elige este servicio para brindarle a su carro el mejor color de pintura que se puede ofrecer para el cuidado de su vehiculo al color de su gusto.

Alineacion

Elige este servicio para alinear tu vehiculo para arreglar la alineacion de tus ruedas y llantas para que estas puedan trabajar de forma pararela.

Chequeo y Reparacion

Elige este servicio para que se pueda chequear los posibles problemas que tenga su vehiculo y poder arreglar esos problemas menores que puedan existir.

Limpieza de inyectores

Elige este servicio para limpiar los inyectores de combustible de su vehiculo para el mejor funcionamiento en la potencia.

Limpieza del sistema de Refrigeracion

Elige este servicio para que se limpie el sistema de refrigeracion debido a que el 40% de los problemas del motor se debe a la falta de mantenimiento de esta.

Cambio de liquido de frenos

Elige este servicio para que nosotros cambiemos el liquido de los frenos , esto ademas de ser parte importante para la seguridad y el funcionamiento del carro se debe cambiar cada cierto tiempo.

Cambiar aceite de transmision

Elige este servicio para el mantenimiento de su vehiculo para cambiar al aceite necesario que requiere el vehiculo para un optimo y buen funcionamiento.

Cambio de lubricante para engranajes

Elige este servicio para lubricar los engranajes y aumentar la vida util del vehiculo al remorcar los objetos, ademas de ayudar a su resistencia y aumentar su ahorro de combustible.
